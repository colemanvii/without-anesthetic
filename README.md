# Without Anesthetic
> Part of the Haven Personal Operating System

A clean, well-lighted place for a single text. This is a deliberate rejection of the "feed" in favor of the "vow."

## Philosophy
- **Quiet Interface:** No tracking, no comments, no noise.
- **Stark Reality:** Designed for reading the things that usually make us want to look away.
- **Southern Futurist:** High-contrast, monospace precision, and timeless serif typography.

## Technical Details
- **Architecture:** Single-file HTML/CSS/JS.
- **Deployment:** Automated via GitHub Actions.
- **Features:** - Dynamic Reading Time calculation.
  - LocalStorage-backed Dark/Light modes.
  - Keyboard shortcuts (`T` for Theme, `C` for Copy).
